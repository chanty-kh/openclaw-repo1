# openclaw-repo1

A simple personal website for Ty Chan.

## Tech

This site uses:
- HTML
- CSS

No framework is required, so it is easy to understand and easy to edit.

## Files

- `index.html` – main website page
- `styles.css` – styling

## How to run locally

Just open `index.html` in a browser.

## Customize

You can update:
- your name
- about section
- projects
- contact email
- colors and layout in `styles.css`
